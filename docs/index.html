Go-Boiler ?

Go-Boiler is a combination of existing libraies and hand crafted function required to create website/webapps. Its not a full fledge framework having orm or caching or even its own routing.

Go-boiler is built-on top if some of the best golang libraries is out there ( with certain wrapper functions over it to make it more fancy).

Features

Go-boiler is built-on top of existing libraries, but still it has certains thing those are original. For example static routing and response generator.

  • Routing
    • Static Routes
    • Dynamic Routes
  • Response Rendering
    • HTML Rendering
    • Text Rendering
    • JSON Response
  • CSRF Security

Installation


You need source code of go-boiler in your machine. You can either clone the project from github or download source code.

For clonning project
$ git clone https://www.github.com/mchampaneri/go-boiler.git

Once you have source code in your machine, you are ready to statr developement work.

Getting Started

Before you can see your appliaction you have to build server for your appliaction. Once you build server you will be able to see sample page.

    Build server
  • Open Terminal / CMD
  • Goto root folder of your project
    $ application\ :
  • Travers to server-logic folder
    $ application\server-logic\ :
  • Generate server in parent fodler
    $ appliaction\server-logic\ : go build -o ../xerver
    Start server
  • Open Terminal / CMD
  • Goto root folder of your project
    $ application\ :
  • Start xerver executable
    $ appliactions\: ./xerver

After starting server open http://localhost:8081 in browser. You should see welcome page.
