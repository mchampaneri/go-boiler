Introduction

Go-boiler plate is created using combining existing (golang & node) libraries out there. Libraries are not choosen for best performance, instead they are selected by balancing between performance and ease of use and learning curve.

It mostly inspired from laravel framework ( php ). Event it uses same laravel-mix for frontend developement and preprocess js and css.
